# CYBERFORGE - Интернет-магазин кастомных ПК

## Структура проекта

```
src/
├── app/
│   ├── layout.tsx          # Корневой layout с метаданными
│   ├── page.tsx            # Главная страница
│   └── globals.css         # Глобальные стили (киберпанк тема)
├── components/
│   ├── layout/
│   │   ├── Header.tsx      # Шапка с навигацией и корзиной
│   │   └── Footer.tsx      # Футер с контактами
│   ├── sections/
│   │   ├── HeroSection.tsx     # Главный экран с анимациями
│   │   ├── CatalogSection.tsx  # Каталог с фильтрами
│   │   └── FeaturesSection.tsx # Секция преимуществ
│   ├── product/
│   │   └── ProductCard.tsx     # Карточка товара
│   └── configurator/
│       └── ConfiguratorSection.tsx  # Интерактивный конфигуратор
├── store/
│   └── index.ts            # Zustand stores (корзина, конфигуратор)
├── types/
│   └── product.ts          # TypeScript типы
└── data/
    └── products.ts         # Mock данные продуктов и комплектующих
```

## Реализованные функции

### 1. Главная страница (Hero Section)
- Полноэкранный баннер с анимированными эффектами
- Анимированный заголовок "Создай машину своей мечты"
- CTA кнопки "Собрать ПК" и "Смотреть каталог"
- Секция преимуществ (гарантия, тестирование, доставка, сборка)
- Декоративные элементы ПК (SVG-схема)

### 2. Каталог (Catalog Section)
- Сетка товаров (Grid layout)
- Карточки товаров с фото, названием, характеристиками, ценой
- Бейджи (NEW, скидка, тип ПК)
- Фильтры сбоку:
  - По назначению (Игровой, Рабочая станция, Для творчества, Офисный)
  - По цене (слайдер)
  - По производителю видеокарты (NVIDIA, AMD, Intel)
  - По минимальной RAM
- Сортировка (популярность, цена, рейтинг, новинки)
- Адаптивный дизайн

### 3. Конфигуратор (Configurator)
- Интерактивная форма выбора комплектующих
- Категории: Процессор, Видеокарта, ОЗУ, Накопитель, Корпус, Охлаждение, БП, Мат. плата
- Поиск по компонентам
- Реальное время обновления цены
- Индикатор совместимости с предупреждениями
- Прогресс-бар сборки
- Добавление кастомной сборки в корзину

### 4. Карточка товара (ProductCard)
- Фото товара с placeholder
- Название и краткое описание
- Рейтинг и количество отзывов
- Краткие характеристики (CPU, GPU, RAM)
- Цена и скидка
- Кнопки "Купить" и "Избранное"
- Hover эффекты

### 5. Header
- Фиксированная шапка с glassmorphism эффектом
- Навигация с выпадающим меню каталога
- Корзина с SidePanel
- Мобильное меню

### 6. Footer
- Контакты
- Ссылки на разделы
- Социальные сети
- Подписка на рассылку

## Дизайн и стиль

### Цветовая гамма (Dark Mode)
- **Фон**: Глубокий черный (#0a0a0a)
- **Карточки**: Темно-серый (#121212)
- **Акценты**: 
  - Неоновый голубой (#00f0ff)
  - Фиолетовый (#8b5cf6)
  - Зеленый (#00ff88)

### Эффекты
- Glassmorphism (размытие + прозрачность)
- Неоновое свечение (glow)
- Градиентные границы
- Сетка на фоне (cyber-grid)
- Плавные анимации Framer Motion

## Технологии
- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS 4**
- **shadcn/ui** компоненты
- **Framer Motion** для анимаций
- **Zustand** для state management
- **Lucide Icons**
